Module Number One: Computer Animation

  • In this module I worked with the program Animation Works, a program copyrighted in 1991 with all the glory and technology that was available five years ago. I worked with version 1.00.00, which can never be a good thing, as the first released version of a program (though not titled as such) could be concidered the beta test program . These are used so that the programmers can get feedback from customers about how well the program works, and discover possible glitches. With this slightly corrupted program, I happened to create a simple animation with music and sound, involving a virus, bomb, mass amounts of blood, and Wally. This exact program will be of no technological use in the future, as it is already five years old. Though that might not sound old, concider this : in 1991, the fastest PC computer was a roaring 486/33Mgz with 8Mb of RAM. By todays standards, that couldn't run a simple calculator program. The field of computer animation, though, has allowed people to create 3D images, that can be used in cartoons, simulations, and commercials for technical colleges that advertise on TV. Animation Works, once one has figured out how to work around the eminent problems, can be quite powerful, and one can create impressive animations without ease. This station should introduce the user to the technoloy of 3D rendering, for that is the direction much computer animation is leaning towards.

Module Number Three: Graphic Design

  • In this module I used a Macintosh computer and Canvas to create graphics. Canvas appears to be a rather powerful graphical art program, but the Mac creates a balance and maintains the constant of only so-so useful and user-friendly programs. I downloaded a graphical file off of the Internet, changed the colors around, enlarged it, and printed it on a shirt. That was fun! I doubt I have a future in seamstressing, but the expirience with a computer art program could help a future in advertising, among other things. I liked the fact that I could exercise absolute control over T-Shirt printing in a normally democratic society. There is not much room for improvement in this module, because you will get out of it what you put in.
